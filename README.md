# IoT and Edge AI

**Typst** source code for a presentation and workshop series on **IoT and Edge AI**.

## Contents
- **ESP32** — HTTP REST control, WebSocket communication, LoRa
- **Edge Impulse** — TinyML deployment
- **YOLO on Raspberry Pi** — real-time object detection at the edge
- **Mosquitto** — MQTT broker setup
- **Node-RED** — flow-based IoT automation, dashboard widgets
- **InfluxDB** — time-series storage for sensor and IoT data
- **Prometheus / Grafana** — monitoring stack

> [!NOTE]
> All slides are written in [Typst](https://typst.app) using the `touying` package.
